About

This has been a personal project to learning image recogintion implementation. It takes a given image or artwork and tries to predict which Magic: The Gathering colour it might be.

It was built using transfer learning of the MobileNet provided by Keras. It's trained and validated on a set of 2,600+ magic card artworks pulled from Scryfall API.
